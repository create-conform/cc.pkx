# cc.pkx

Library for loading PKX modules, and working with PKX packages.

## package.json

Version 0.2 Sample:

```javascript
{
    "name": "cc.demo",
    "version": "0.0.2",
    "pkx" : {
        "main": "demo.js",
        "dependencies": [
            "cc.config.0.2",
            "cc.host.0.2"
        ]
    }
}
```
